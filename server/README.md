# ShopLoyalty Server side code

## Setup env file

File name: `.env`

```txt
DATABASE_URL = YOUR DATABASE URL
JWT_SECRET = SECRET CODE
JWT_EXPIRES_IN = TIME AS NUMBER
PORT = PORT AS NUMBER
SALT_ROUNDS = SALT AS NUMBER
```

## For start use:

```bash
npm run start:dev
```
